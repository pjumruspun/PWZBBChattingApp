# Parallel and Distributed System (Not-so) Mini Project
## By PWZBB Team

## Tools Used
- Frontend: React
- Backend: Nestjs
- Load Balancer: Nginx
- Docker

## Ports
- Frontend: 3000
- Backend: 3001
- Load Balancer: 8080

## Command
### Starting the project
- docker-compose up
### Stopping the project
- docker-compose down
